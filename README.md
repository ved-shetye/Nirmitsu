# Nirmitsu Design Labs

Industrial Additive Manufacturing, 3D Modelling, and Design Solutions website.

## Features
- Fully responsive design built with HTML, CSS, and Bootstrap.
- Service cards, modern typography, custom vector SVGs, and animations.
- Configured for Vercel deployment with clean URL routing.

## Development
To run locally:
```bash
node server.js
```
Open [http://localhost:8080](http://localhost:8080) in your browser.
